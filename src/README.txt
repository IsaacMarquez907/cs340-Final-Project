--------------------Steps to download SQL tables and get website running------------------

1.Import 'rateMyStudentsSQL.sql' into your phpMyAdmin
	-NOTE: There is a Student, Professor, Student, Term, Class, Takes, and Has table. 
2.Edit config file with your username and password for phpMyAdmin
3.Put public, views, index.js, and config.js in a directory
4.Index.js is the server File
4.Run with either forever or just npm, your choice.
